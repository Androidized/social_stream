= {Social Stream}[https://github.com/ging/social_stream] 

Social Stream is a framework for building social network websites (engine for Ruby on Rails). 
It provides a robust and flexible framework with {social networking}[http://en.wikipedia.org/wiki/Social_network] features and {activity streams}[http://activitystrea.ms/] for building websites.
